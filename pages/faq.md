### Android Development Questions

**Q: What version of Android should I target?**

A: We'll target Android 10 (API 29) as minimum and Android 13 (API 33) as target. Check course materials for any updates.

**Q: Should I learn Kotlin or Java?**

A: We'll use both! The course starts with Java and introduces Kotlin midway through. Both are important for Android development.

**Q: What's the difference between an Activity and a Fragment?**

A: Activities are full-screen components; Fragments are reusable UI pieces within an Activity. You'll learn the details in Week 4.

**Q: Where can I find more Android development help?**

A: Check the [Resources]({{ '/pages/resources/' | relative_url }}) page for links to Stack Overflow, official documentation, and more.

### Troubleshooting

**Q: Android Studio won't start. What do I do?**

A: 
1. Make sure Java JDK 17+ is installed
2. Try deleting the .gradle folder in your home directory
3. Reinstall Android Studio
4. Contact the instructor if problems persist

**Q: My app keeps crashing. How do I debug?**

A: Check the Logcat in Android Studio for error messages. The [Resources]({{ '/pages/resources/' | relative_url }}) page has debugging guides.

**Q: The emulator is very slow. What can I do?**

A: Enable virtualization in your BIOS settings (for Windows), or use Android Studio's hardware acceleration features.
